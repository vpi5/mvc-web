## 项目启动、配置说明
依赖于webpack4 babel解析器等

#### 安装依赖说明
````
    yarn install || npm install
````

#### 项目启动说明
```
    yarn start --[启动文件名称]

    启动文件名称：
    startUpConfig/*.js

    .
    ├── startUpConfig
    │   ├── ALL.js  ## [Test + Test1]
    │
    ├── END！ 
```

#### 项目打包说明
```
    yarn build [参数1]
    
    参数1：webpack 入口文件名 详见上方启动文件名
```
